# Social Connections App
## Overview
*This App is in development*

This is an android app with which you can connect to a friend using NFC and adding them to your local graph.
See how many friends you have in common, or if you are only distantly acquainted.

### Startup

Upon first startup you are prompted with a simple screen asking for your username. This cannot be changed, so choose wisely!
After entering your chosen name your keypair is generated, and you are brought into the Main Menu.

In the Main Menu, you are encountered with multiple different buttons: The Social Connection section, the Add Friend section, and the Seed Debug Graph button.
In either section there are two buttons, one for the host, and the other for the client. One Device may start hosting and the other can then find the host via NFC.

### The Social Connection Section

These buttons are to display the minimum distance between each-other. Do you know each-other over common friends? Family? 
This works up to a distance of 4! 

### The Add Friend Section

This section allows you to add the other peer as a friend to you. This will reduce your minimum distance from the previous section to One (1) and it will automatically update 
both of your friends lists that you've added a new friend!


### Seed Debug List

Would you simply like to test the features of the app?
That's where the Seed Debug List comes in! With this a friend group is assigned to your user and upon connection you and the other peer can see the generated graph.
Both users need to use this Debug List to display it.)
As the default we have the maximum possible distance between two users.
